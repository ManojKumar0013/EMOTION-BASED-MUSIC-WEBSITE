ℹ️ About NexTune

Where emotion meets music

🎯 Project Overview

NexTune is an Emotion-Based Music Recommendation Website developed as an end-semester project for User Interface Design at Amrita School of Computing, Amaravati.

Select how you feel — happy, sad, energetic, calm — and NexTune instantly surfaces the perfect playlist using the Jamendo API (600 000+ free, legal tracks). The background reacts to your mood with live animations (rain, fire, breathing rings, floating hearts) inspired by the film Inside Out.

🛠 Technologies Used

  • HTML5 Semantic structure & accessibility
  • CSS3 Glassmorphism, animations, media queries
  • JavaScript Web Audio API, Fetch, LocalStorage
  • Jamendo API 600K+ free songs
  • Web Audio API Real-time music visualiser
